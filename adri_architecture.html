Arquitectura General

Sistema multi-agente con supervisor orquestador

Definición
ADRI (Asistente Digital de Respuesta Inteligente) es un sistema multi-agente que utiliza Claude 3.5 Sonnet como motor base para orquestar 10 agentes especializados mediante delegación inteligente y combinación de resultados.

Memoria Persistente

-- Estructura PostgreSQL + pgvector CREATE TABLE user_memory ( id SERIAL PRIMARY KEY, user_id VARCHAR(255), content TEXT, embedding vector(3072), metadata JSONB ); -- Búsqueda por similitud SELECT * FROM user_memory ORDER BY embedding <=> query_vector LIMIT 5;
